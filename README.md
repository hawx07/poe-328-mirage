# ⚔ Path of Exile 3.28 — Mirage Patchnotes Breakdown

> A fan-made visual patchnotes breakdown and build guide for **Path of Exile: Mirage (3.28)**

🔗 **Live Site → [hawx07.github.io/poe-328-mirage](https://hawx07.github.io/poe-328-mirage/)**

---

## Features

- **Ascendancy Changes** — All 7 changed ascendancies with buff/nerf/change indicators
- **Gem Changes** — 30+ skill gems: new holy skills, buffs, nerfs, reworks, new transfigured variants
- **General Changes** — Atlas rework, INT/ES nerf, Exceptional Support Gems, Harbinger removal, and more
- **Build Guides** — 8 detailed builds including meta picks and off-meta sleeper builds
- **PoE Aesthetic** — Dark ornate theme matching the in-game style, with circular icon frames
- **Tooltips** — Hover any icon for a detailed summary of the change
- **Expandable Build Cards** — Click "Show Details" for full playstyle, pros/cons, gem links & key items
- **Responsive** — Works on desktop and mobile
